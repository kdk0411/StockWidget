# StockWidget

윈도우 우측 하단에 주식 시세를 띄워주는 데스크톱 위젯입니다.

- **코너 위젯**: 화면 우측 하단(트레이 위)에 반투명 막대로 **최대 2종목** 표시 (현재가 + 등락률)
- **트레이 아이콘**: 작업표시줄 우측 알림영역에 상주. 우클릭 메뉴로 제어
- **자세히 보기**: **최대 4종목** + 평단가 + **수익률** 표시 (위젯 클릭 또는 트레이 클릭)
- **설정**: 종목 4개(종목 코드)와 평단가, 갱신 주기 입력
- 시세는 기본 **2초**마다 갱신

---

## 필요 환경

- Windows 10 / 11
- [Node.js](https://nodejs.org/) 18 이상 (LTS 권장)

## 설치 (소스에서 받기)

```powershell
git clone https://github.com/<사용자명>/stock-widget.git
cd stock-widget
npm install        # 의존성 설치 (최초 1회)
```

> 이미 받은 폴더(`stock-widget`)가 있다면 그 안에서 `npm install` 만 한 번 실행하면 됩니다.

## A. 패키징 없이 바로 쓰기

설치파일(.exe)을 만들지 않고, 소스 폴더에서 그대로 실행하는 방법입니다.
**평소 사용은 이 방법이 가장 간단합니다.**

### A-1. 더블클릭으로 실행 (가장 쉬움)

`stock-widget` 폴더 안의 파일을 더블클릭하세요:

| 파일 | 설명 |
|------|------|
| **`실행.vbs`** | 추천. 콘솔창 없이 조용히 실행 (트레이 위젯에 적합) |
| **`실행.bat`** | 검은 콘솔창이 함께 뜨는 버전 (로그/에러 확인용, 창 닫으면 종료) |

> **바탕화면에 두고 싶다면**: `실행.vbs` 우클릭 → [바로 가기 만들기] → 만들어진 바로가기를 바탕화면으로 옮기세요.
>
> 두 런처 모두 이 PC의 `ELECTRON_RUN_AS_NODE` 환경변수를 자동으로 비활성화한 뒤 실행합니다.
> (이 변수가 살아있으면 Electron이 GUI 대신 Node로 떠버립니다.)

### A-2. 터미널에서 실행 (개발 모드)

코드를 수정하며 테스트할 때 사용합니다.

```powershell
cd stock-widget
$env:ELECTRON_RUN_AS_NODE=""    # 현재 세션만 환경변수 해제
npm start
```

## B. 패키징해서 .exe 만들기 (배포용)

다른 PC에 **Node.js 설치 없이** 전달하고 싶을 때, 설치파일(.exe)로 묶는 방법입니다.

빌드 명령은 `npm run dist` 한 줄이지만, electron-builder가 코드서명 도구(`winCodeSign`)
압축을 풀 때 **윈도우의 symlink(심볼릭 링크) 생성 권한**이 필요합니다.
일반 권한으로는 아래 에러가 나며 실패합니다:

```text
Cannot create symbolic link ... 클라이언트가 필요한 권한을 가지고 있지 않습니다
```

이를 해결하는 **두 가지 방법** 중 하나를 쓰면 됩니다.

### 방법 1. 개발자 모드 켜고 빌드 (권장 · 한 번만 설정)

1. **개발자 모드 켜기**
   `설정 → 업데이트 및 보안 → 개발자용 → "개발자 모드" 켜기`
   (개발자 모드를 켜면 관리자 권한 없이도 symlink 생성이 허용됩니다.)
2. **빌드** — 프로젝트 폴더에서 실행:

```powershell
cd stock-widget
$env:ELECTRON_RUN_AS_NODE=""    # 이 PC 환경변수 해제 (안 그러면 빌드 도구가 오작동)
npm run dist
```

한 번 개발자 모드를 켜두면, 이후엔 `npm run dist` 만 다시 실행하면 됩니다.

### 방법 2. 관리자 권한 PowerShell로 빌드

개발자 모드를 켜기 싫다면, **관리자 권한 PowerShell**에서 빌드하면 됩니다.

1. 시작 메뉴에서 **PowerShell** 검색 → 우클릭 → **"관리자 권한으로 실행"**
2. 프로젝트 폴더로 이동 후 빌드:

```powershell
cd "C:\...\stock-widget"      # 실제 경로로 변경
$env:ELECTRON_RUN_AS_NODE=""
npm run dist
```

관리자 권한이면 symlink 생성이 허용되어 빌드가 통과합니다.

### 빌드 결과물

`dist/` 폴더에 다음이 생성됩니다.

- **`StockWidget Setup x.x.x.exe`** — 배포용 설치파일 (이 파일만 전달하면 됨, ~75MB)
- **`win-unpacked/`** — 설치 없이 폴더째 복사해서 쓰는 포터블 버전 (안의 `StockWidget.exe` 실행)

사용자 PC에 별도 런타임 설치 없이 그대로 설치/실행됩니다.

> 코드 서명을 하지 않았으므로, 다른 PC에서 처음 실행 시 SmartScreen 경고가 뜹니다.
> → **추가 정보 → 실행** 을 누르면 정상 실행됩니다.
>
> 버전을 올리려면 `package.json` 의 `"version"` 을 수정하고 다시 `npm run dist` 하세요.

## 사용법

1. 앱 실행 → 트레이에 초록색 차트 아이콘 등장, 우측 하단에 위젯 표시
2. 트레이 **우클릭 → 설정** → 종목과 평단가 입력 후 저장
   - 종목 입력: 종목 코드(예: `A035420`)
   - 평단가를 비우면 수익률 대신 당일 등락률만 표시
3. 위젯 클릭(또는 트레이 클릭) → **자세히 보기**로 4종목 + 수익률 확인
4. 트레이 메뉴에서 위젯 표시/숨김, 종료 가능

## 설정 저장 위치

`%APPDATA%\StockWidget\config.json`
